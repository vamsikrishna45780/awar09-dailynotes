IAM Policy : A Document that contains set of permisisons on AWS environment, Written in JSON format. 

AWS Managed : AWS Provides this based on the service and service use cases. 

AWS Managed - Job Function : AWS Provides this based on current job roles, and they required permisisons to perform that specific job. i.e; dba, readonly, administrator, SecurityAudit, SupportUser.. 

Customer Managed Policy : For custom requirements, based on customer use cases.. We can depend on Customer managed policy.
